# LRCS Press Style

TeX/LaTeX style files for LRCS Press publications. :contentReference[oaicite:0]{index=0}

## Overview

This repository contains the official style files used to format publications for **LRCS Press** — the publishing arm of the Lastoria Royal College of Science. These files help authors consistently prepare documents (e.g., articles, reports, books) in the required TeX format.

- **`.sty`** files — LaTeX style definitions (e.g., `lrcsstyle.sty`) :contentReference[oaicite:1]{index=1}
